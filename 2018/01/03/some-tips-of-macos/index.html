autojump–按权重直达目录

使用： brew install autojump

在~/.zshrc中添加[ -f /usr/local/etc/profile.d/autojump.sh ] && . /usr/local/etc/profile.d/autojump.sh

source ~/.zshrc

基本原理：autojump的工作方式很简单，它会在你每次启动命令时记录你当前位置，并把它添加进它自身的数据库中。这样，某些目录比其它一些目录添加的次数多，这些目录一般就代表你最重要的目录，而它们的“权重”也会增大。

用法：

autojump官网使用文档

j foo 跳转到数据库中路径包含foo的目录，支持tab补全

jc bar 跳转到当前目录下的子目录

jo dirname 用文件管理器打开dirname目录

jco child-dir Opening a file manager to a child directory

j 进入当前权重最高的目录

j --stat 查看当前权重

j -a [dir] 访问目录数据库，并修改它的内容

j -i [weight] 增加当前目录权重

j -d [weight] 减少当前目录权重

j -s 跟踪所有改变，显示数据库中的统计数据

j --purge 命令将会把不再存在的目录从数据库中移除
